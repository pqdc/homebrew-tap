# pqdc Homebrew Tap

Homebrew tap for the pqdc governance toolkit — tools that make AI
collaboration auditable and trustworthy.

## Setup

```
brew tap pqdc/tap
```

## Available Tools

| Tool | Install | Description |
|------|---------|-------------|
| pqcc | `brew install pqdc/tap/pqcc` | Governance dashboard for pqdx state files |

More tools shipping soon: pqsn, pqit, pqgd, pqsl, pqvl, pqsq, pqfg.

## Quick Install

```
brew install pqdc/tap/pqcc
```
